📊 Centralisation des logs avec rsyslog

Mise en place d’un serveur de centralisation des journaux système à l’aide de rsyslog, intégré à un environnement Active Directory. Cette solution permet de collecter, organiser et analyser les logs de plusieurs machines afin d’améliorer la supervision, la sécurité et le diagnostic des incidents.

  • Déploiement d’un serveur rsyslog sous Linux
  • Intégration au domaine Active Directory
  • Centralisation et structuration des logs par machine, service et date
  • Catégorisation des journaux (security, system, web)
  • Génération de logs structurés au format JSON

🖥️ Déploiement automatisé avec FOG Project

Installation et configuration d’un serveur FOG Project sous Ubuntu Server afin d’automatiser la capture et le déploiement d’images système via le démarrage réseau PXE. Cette solution permet de standardiser les postes et de réduire le temps de déploiement dans une infrastructure professionnelle.

  • Installation de FOG Project sous Ubuntu Server
  • Configuration du démarrage réseau PXE (DHCP, TFTP)
  • Capture et déploiement d’images système
  • Administration centralisée via l’interface web FOG
  • Gestion des incidents PXE et sécurisation des accès
